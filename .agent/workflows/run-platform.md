# كيفية تشغيل المنصة التعليمية

لتشغيل المنصة بالكامل، تحتاج إلى فتح نافذتين للتبويب (Terminal) وتشغيل الأوامر التالية:

### 1. تشغيل السيرفر (Backend)

افتح نافذة Terminal جديدة وانتقل إلى مجلد الـ backend:

```bash
cd backend
# في حال واجهت مشكلة في الصلاحيات (PowerShell):
cmd /c npm run dev
# أو بشكل عادي:
npm run dev
```

### 2. تشغيل الواجهة (Frontend)

افتح نافذة Terminal ثانية وانتقل إلى مجلد الـ frontend:

```bash
cd frontend
# في حال واجهت مشكلة في الصلاحيات (PowerShell):
cmd /c npm run dev
# أو بشكل عادي:
npm run dev
```

### 3. فتح الموقع

بعد تشغيل الأمرين بنجاح، افتح المتصفح على العنوان التالي:
[http://localhost:3000](http://localhost:3000)
